# Automation of a stores daily sales records
This code is for store owners, it allows them to store the  sales of the day and have a resume of them showing names, amounts of products, amounts of sales and the total of the sales of all day.
The objective of this project is to apply fundamental programming concepts such as modular programming, use of tuples, dictionaries, and lists.

This program was made in python language

First, an analysis of the client's needs was carried out. Once these had been identified, a flowchart was drawn up to help understand the logic behind our system.
